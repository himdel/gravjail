gravjail
===============

Entry in PyWeek #11  <http://www.pyweek.org/11/>
Team: gravjail
Members: himdel, clime
Authors: Martin Hradil, Michal Novotný
(provisional) graphics by himdel


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/
  PyODE:      http://pyode.sourceforge.net/



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Rotate the ship using left and right arrow keys, accelerate using the up arrow key. deccelerate using the down arrow.
For additional players, use W,A,S,D, I,J,K,L and 8,4,5,6.



GETTING NEW VERSION:

Current versions (post-pyweek) will be at http://github.com/himdel/gravjail .
